# New York Time Articles with Next.js

## About

New York Times Articles (API) page with basic auth using Next.js and Node

## What did I use?

* React - Next.js
* SCSS
* GraphQL
* Node
* JWT
* Heroku
* Mongo (Mongoose)
* Express.js 
* Node.js
* Axios
* Cookies
